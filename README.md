# ESP32-S3 × PlatformIO（VSCode）IoT入門テキスト

本リポジトリは、**VSCode + PlatformIO を使用し、ESP32-S3 でIoTシステムを開発するための学習用テキスト教材**です。
主に **マイコンプログラミング初学者〜IoT基礎習得レベル** を対象としています。

---

## 教材の特徴

* VSCode + **PlatformIO 対応**
* 無線マイコン **ESP32-S3** を使用
* GPIO制御（LED / スイッチ）から丁寧に解説
* Wi-Fi通信・Webアプリケーション開発に対応
* TCP / MQTTなどのネットワーク通信も扱う
* 授業・研修・自主学習向けに構成

---

## 対象者

* C / C++ の基本文法を学習済み、または学習中の方
* マイコン・組み込み開発をこれから始めたい方
* IoTシステム開発に興味がある方
* PlatformIO を使った実践的な教材を探している方

---

## 内容（例）

* IoTと無線マイコンの基礎
* ESP32の仕様と開発環境構築
* GPIO制御（LED / スイッチ）
* LCD表示・シリアル通信
* PWM / 割込み処理
* Wi-Fi通信（Station / SoftAP）
* Webサーバ構築とブラウザ操作
* ファイルシステム（LittleFS）
* TCP通信 / MQTT通信

※ 上記は一例です。詳細な教材内容は本文をご参照ください。
